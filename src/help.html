What is a set?

Set is a game of pattern recognition. There are several ways to play, including singleplayer, pvp, co-op and timed mode.

(Most of these modes remain to be implemented.)

The goal is to deal some cards onto the table, and find three of them that match a certain criterion.

Each card has 4 attributes : shape, color, shading and number.

We call 3 cards a "set", if in each of these 4 attributes, the 3 cards either have the same value, or all different values.

Some python code for determining whether three cards form a "set" might look as follows:

def is_set(card1, card2, card3):

    shapes = set([card1.shape, card2.shape, card3.shape])

    colors = set([card1.color, card2.color, card3.color])

    shadings = set([card1.shading, card2.shading, card3.shading])

    numbers = set([card1.number, card2.number, card3.number])

    return not (len(shapes)==2 or len(colors)==2 or len(shadings)==2 or len(numbers)==2)
